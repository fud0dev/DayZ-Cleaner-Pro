🛠️ DayZ Cleaner Pro
DayZ Cleaner Pro es un script de automatización ligero en formato Batch (.bat) diseñado para optimizar el rendimiento de DayZ eliminando archivos temporales, registros de errores y volcados de memoria innecesarios que pueden acumularse y ocupar gigabytes de espacio.

✨ Características
Interfaz de Usuario Estilizada: Menú visual basado en terminal técnica para una mejor experiencia.

Limpieza de Logs (.RPT): Elimina los reportes de servidor y cliente que crecen indefinidamente.

Purga de Crash Dumps: Borra los archivos .bidump y .mdmp generados tras cierres inesperados.

Limpieza de Scripts: Elimina archivos temporales de scripts corruptos.

Seguro y Ligero: No toca archivos esenciales del juego ni configuraciones de usuario (.cfg).

🚀 Instalación y Uso
Descarga el archivo DayZ_Cleaner.bat.

Colócalo en cualquier carpeta (no es necesario que esté en la carpeta del juego).

Ejecuta el archivo haciendo doble clic.

Selecciona la opción [1] en el menú para iniciar la limpieza.

📂 Archivos que se eliminan
El script actúa sobre la ruta %LOCALAPPDATA%\DayZ\, eliminando:

*.rpt (Reportes de sistema)

*.log (Registros de sesión)

crash_* (Informes de errores)

script_* (Temporales de ejecución)

⚠️ Nota de seguridad
Este script solo elimina archivos temporales. No borra tu personaje, ni tus progresos en servidores, ni tus configuraciones de teclas.
